Contributing to DubbiOvi
First of all, thank you for your interest in contributing to DubbiOvi.
DubbiOvi is an open-access research software project developed at the University of Oviedo (Spain) with the aim of making AI-assisted audiovisual translation accessible to education, research, and professional practice.
Every contribution—large or small—is greatly appreciated.
 
Ways to Contribute
You can contribute in many different ways, including:
•	reporting bugs;
•	suggesting new features;
•	improving documentation;
•	correcting typos;
•	translating documentation;
•	improving accessibility;
•	testing new releases;
•	contributing code.
Not every contribution needs to involve programming.
 
Before You Start
Please:
•	read the User Manual;
•	review existing GitHub Issues;
•	check whether your suggestion has already been discussed.
If you are planning a major contribution, we recommend opening an Issue first to discuss your proposal.
 
Reporting Bugs
When reporting a bug, please include:
•	DubbiOvi version;
•	operating system;
•	steps to reproduce the issue;
•	expected behaviour;
•	actual behaviour;
•	screenshots or error messages where possible.
Clear reports help us resolve issues more quickly.
 
Suggesting Features
Feature requests are welcome.
Please explain:
•	the problem you are trying to solve;
•	your proposed solution;
•	possible alternatives;
•	expected benefits for users.
 
Code Contributions
If you would like to contribute code:
1.	Fork the repository.
2.	Create a dedicated branch.
3.	Make your changes.
4.	Test your modifications.
5.	Submit a Pull Request.
Please keep Pull Requests focused on a single topic whenever possible.
 
Coding Principles
We encourage contributions that are:
•	readable;
•	well documented;
•	modular;
•	maintainable;
•	compatible with existing project architecture.
Whenever possible, new functionality should include appropriate documentation.
 
Respect the Project Philosophy
DubbiOvi is committed to:
•	openness;
•	accessibility;
•	transparency;
•	privacy;
•	educational value;
•	reproducible research.
Contributions should align with these principles.
 
Code of Conduct
By participating in this project, you agree to follow the project's Code of Conduct.
Please treat everyone with respect and professionalism.
