# DubbiOvi

### Open Audiovisual Translation Environment

**An open-access, AI-assisted platform for audiovisual translation, developed at the University of Oviedo (Spain).**

**Website:** https://dubbiovi.com
**DOI:** https://doi.org/10.5281/zenodo.20683887
**Current Release:** v1.3.7 Academic Edition
**License:** MIT License

---

# Overview

DubbiOvi is an open-source, AI-assisted audiovisual translation (AVT) environment designed for **education, academic research, and professional workflows**.

Developed at the **University of Oviedo**, DubbiOvi provides a local-first workspace for transcription, dialogue segmentation, translation, terminology management, project organisation, and document export.

The platform has been created to support the teaching and practice of dubbing translation while promoting **Open Science**, **responsible AI**, **privacy by design**, and **knowledge transfer**.

# Key Features

## AI-Assisted Workflow

* AI-powered speech transcription using Gemini 2.5 Flash
* Automatic dialogue segmentation into timecoded takes
* AI-assisted translation workflow

## Translation Workspace

* Translation editing interface
* Timeline management
* Original and translated dialogue synchronisation
* Professional take-based workflow

## Terminology

* Built-in glossary management
* Glossary export to CSV, XLSX and JSON

## Project Management

* Local `.dubbiovi` project files
* Save and reopen complete workspaces
* Undo / Redo history
* Workflow status tracking (Pending, Reviewed, Locked)

## Editing Tools

* Merge adjacent takes
* Split takes proportionally
* Repeat selected takes
* Professional playback controls

## Export

* Microsoft Word (.docx)
* Microsoft Excel (.xlsx)
* CSV

# Privacy

DubbiOvi follows a **local-first philosophy**.

Your audiovisual projects remain on your own computer.

The application does **not** include:

* telemetry;
* analytics;
* usage tracking;
* project monitoring;
* automatic cloud storage.

Only users who voluntarily register to download the software provide basic contact information.

For more information, please refer to the project's **Privacy Policy** and **Data Management Plan**.

# Recommended Workflow

### AI Transcription

Gemini 2.5 Flash currently performs best with audiovisual files of approximately **15 minutes or less**.

For longer productions, we recommend:

* processing videos in shorter sections;
* importing existing scripts;
* translating pre-segmented material.

### Translation Without AI

DubbiOvi is fully functional without AI services.

Users may:

* import scripts;
* create dialogue takes manually;
* edit translations;
* organise timing;
* export professional translation documents.
