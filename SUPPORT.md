Support
Thank you for using DubbiOvi!
DubbiOvi is an open-access research software project developed at the University of Oviedo (Spain) to support audiovisual translation, dubbing, and subtitling workflows.
If you encounter a problem or have a question, we are happy to help.
 
Before Contacting Support
Please make sure that you have:
•	installed the latest available version;
•	read the User Manual;
•	checked the project's GitHub Issues to see whether your question has already been answered.
Many common questions are resolved by updating to the latest release.
 
Types of Support
Technical Issues
Examples include:
•	installation problems;
•	application crashes;
•	unexpected behaviour;
•	export failures;
•	AI integration issues.
 
Bug Reports
When reporting a bug, please include:
•	DubbiOvi version;
•	operating system;
•	a short description of the issue;
•	steps required to reproduce it;
•	screenshots or error messages (if available).
 
Feature Requests
Suggestions for new features are always welcome.
DubbiOvi is an evolving research project, and community feedback plays an important role in shaping future releases.
Please describe:
•	the proposed feature;
•	why it would be useful;
•	possible use cases.

GitHub Issues
Whenever possible, technical issues should be reported through the GitHub Issues page.
This allows bugs to be documented, discussed, and resolved transparently.
 
Community
DubbiOvi welcomes feedback from:
•	researchers;
•	lecturers;
•	students;
•	translators;
•	localisation specialists;
•	audiovisual accessibility professionals;
•	open-source contributors.
Constructive feedback helps improve the software for everyone.
 
Project Philosophy
DubbiOvi is developed as:
•	free software;
•	open-access software;
•	research software;
•	educational software.
Our goal is to make AI-assisted audiovisual translation more accessible to education, research, and professional practice.
 
Response Time
Although every request is read, DubbiOvi is maintained as an academic research project.
Support is therefore provided on a best-effort basis, and response times may vary depending on teaching, research, and project commitments.
We appreciate your patience and understanding.
 
Thank You
Thank you for helping improve DubbiOvi.
Every bug report, suggestion, and piece of feedback contributes to making the platform better for the entire audiovisual translation community.
