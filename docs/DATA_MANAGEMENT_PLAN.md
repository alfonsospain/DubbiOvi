DATA MANAGEMENT PLAN (DMP)

Project: DubbiOvi – Open-Access AI-Assisted Audiovisual Translation Platform

Institution: University of Oviedo (Spain)

Project Lead: Alfonso C. Rodríguez Fernández-Peña

Document version: 1.1

Applies to software version: 1.3.7 Academic Edition

Last updated: July 2026

1. Purpose

This Data Management Plan (DMP) describes how data associated with the DubbiOvi project are collected, managed, processed, stored, protected, shared, preserved, and, where appropriate, deleted throughout the project's lifecycle.

The document has been prepared in accordance with the principles of:

Open Science;
FAIR data principles;
Privacy by Design;
Privacy by Default;
data minimisation;
transparency;
responsible research and innovation.

DubbiOvi has been designed as a local-first audiovisual translation platform in which users retain control over their own working materials while only a minimal amount of administrative information is collected for project management purposes.

2. Data Governance
2.1 Data Controller

Administrative registration data are processed under the responsibility of the University of Oviedo, within the framework of the DubbiOvi project.

The Project Lead coordinates the day-to-day administration of the project and oversees the management of registration data exclusively for the purposes described in this document.

2.2 Authorised Personnel

Access to administrative information is limited to authorised project personnel acting within the scope of their responsibilities.

Personnel with authorised access are subject to confidentiality obligations and appropriate organisational and technical security measures.

3. Categories of Data

The DubbiOvi project manages two clearly differentiated categories of information.

3.1 Administrative Data

Users who voluntarily register to download DubbiOvi may provide:

First name
Last name
Email address
Institution
Country
Operating system
Software version
Registration timestamp

These data are processed exclusively for project administration, dissemination, communication, impact assessment, and software maintenance.

Only the minimum amount of personal information necessary to achieve these objectives is collected.

3.2 User-Generated Software Data

DubbiOvi processes audiovisual translation materials locally on the user's own computer.

These materials may include:

audiovisual files;
subtitle files;
translation memories;
terminology databases;
scripts;
translated text;
timing information;
exported projects.

DubbiOvi does not collect, upload, archive, monitor, or analyse these materials.

Users retain full ownership and responsibility for their own project files.

4. Data Collection Principles

The project applies the following GDPR principles throughout the software lifecycle:

lawfulness;
fairness;
transparency;
purpose limitation;
data minimisation;
accuracy;
storage limitation;
integrity and confidentiality;
accountability.

DubbiOvi has been designed according to the principles of Privacy by Design and Privacy by Default (Article 25 GDPR).

Whenever possible, processing activities are designed to avoid collecting personal information altogether.

5. Data Storage

Administrative registration data are stored using secure institutional Google Workspace services managed by the project administrator.

Access requires institutional authentication and is restricted to authorised personnel.

User-generated project files remain stored exclusively on the user's own devices unless users explicitly decide to employ external storage solutions.

6. Security Measures

Reasonable technical and organisational safeguards are implemented to protect administrative information, including:

authenticated access;
password-protected institutional accounts;
encrypted HTTPS communications;
regular software updates;
secure version control through GitHub;
regular backups of administrative records where appropriate;
restricted administrative permissions;
confidentiality obligations for authorised personnel.

Users remain responsible for protecting the files stored on their own systems.

7. External Artificial Intelligence Services

Some optional DubbiOvi functions allow communication with third-party Artificial Intelligence providers selected by the user.

DubbiOvi acts solely as an interface for these requests.

The project does not permanently retain:

prompts;
translated content;
generated outputs;
audiovisual materials exchanged with external AI services.

Users remain responsible for ensuring that the use of third-party AI services complies with their institutional and legal obligations.

8. Data Sharing

Administrative personal information is never sold, licensed, rented, or disclosed to third parties for commercial purposes.

Only aggregated and fully anonymised statistical information may be used in:

scientific publications;
conference presentations;
institutional reports;
funding applications;
impact assessments;
knowledge-transfer activities.

Individual users cannot be identified through these statistics.

9. Data Retention and Disposal

Administrative information is retained only for as long as necessary to:

administer the project;
communicate software updates;
evaluate dissemination and impact;
satisfy legitimate academic obligations.

When information is no longer required, it will be securely deleted or irreversibly anonymised where appropriate.

Users may request deletion of their personal information at any time.

10. Open Science

DubbiOvi actively supports Open Science through:

open-access software;
transparent development;
public documentation;
version-controlled source code;
reproducible releases via GitHub and Zenodo;
permissive open-source licensing (MIT License).

The project seeks to maximise transparency while respecting users' privacy and institutional data protection requirements.

11. FAIR Principles

The project supports the FAIR principles by ensuring that software outputs and documentation are:

Findable through GitHub and Zenodo.
Accessible as open-access resources.
Interoperable through widely adopted file formats.
Reusable under the MIT License.
12. GDPR Compliance

The DubbiOvi project has been designed in accordance with:

Regulation (EU) 2016/679 (General Data Protection Regulation);
Spanish Organic Law 3/2018 (LOPDGDD);
applicable institutional data protection policies of the University of Oviedo.

Users may exercise their rights of:

access;
rectification;
erasure;
restriction of processing;
objection;
withdrawal of consent where applicable.
